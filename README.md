# Graft
A CS157C (NoSQL) Project:

A simulation of a social media application that runs on a Neo4j graph database. General web structure is built using
Java frameworks (Spring Boot backend, Vaadin frontend) with libraries to handle syncing with the Neo4j Aura instance
running on the cloud. Leverages a public dataset for dummy data to populate the list of users and their relevant details.

Further details to be discussed in project report: https://docs.google.com/document/d/1LaQlv7z_OPqN_r_OchyWCxCeTNpJO1jfdtK6xL62dRU/edit?usp=sharing
